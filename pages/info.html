Información detallada del sistema

El Sistema fue diseñado para que cada formulario cumpla con un rol específico dentro de la gestión de inventario. La aplicación inicia con un formulario de inicio de sesión para posteriormente permitirle al usuario acceder a los distintos módulos del Dashboard. A continuación se muestra la función que cumple cada formulario dentro del Sistema Restock:

Login

Formulario: frmInicio.cs

Función: Permite la validación de las credenciales de acceso.

Descripción: El usuario debe ingresar dentro de los TextBox su nombre de usuario. Luego el programa valida los datos ingresados y permite el acceso al Dashboard.

Dashboard

Formulario: frmDashboard.cs

Función: Sirve como punto de acceso para los demás formularios.

Descripción: Ofrece un menú desplegable con botones para poder navegar entre los distintos formularios.

Clientes

Formulario: frmAgregarClientes.cs, frmGestionarClientes.cs, frmVerClientes.cs

Función: Permite la gestión de clientes dentro de la base de datos.

Descripción: Ofrece un menú desplegable con botones para poder navegar entre los distintos formularios.

Marca y Categoría

Formulario: frmCategoria.cs, frmMarca.cs, frmMarcaCategoria.cs

Función: Permite la gestión de marcas y categorías dentro de la base de datos.

Descripción: El usuario puede agregar, editar, y visualizar las marcas y categorías.

Proveedores

Formulario: frmAgregarProveedor.cs, frmGestionarProveedor.cs, frmVerProveedor.cs

Función: Permite la gestión de proveedores dentro de la base de datos.

Descripción: El usuario puede agregar, editar y visualizar la información de los proveedores.

Stock y Productos

Formulario: frmAgregarProductos.cs, frmGestionarProductos.cs, frmVerProductos.cs

Función: Permite la gestión de marcas y categorías dentro de la base de datos.

Descripción: El usuario puede agregar, editar y visualizar la información del stock y productos.

Usuarios

Formulario: frmAgregarUsuarios.cs, frmGestionarUsuarios.cs

Función: Permite la gestión de los usuarios dentro de la base de datos.

Descripción: El usuario puede agregar y editar la información de los usuarios.
